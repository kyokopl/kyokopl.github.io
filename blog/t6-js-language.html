JavaScript Language

06 July 2018

JavaScript language - describe all the use cases for each:

Parentheses () - can be used to execute a function function callFunc(), to surround conditional statements if (condition), and for grouping in terms of order if ((1*5) + 6).

Brackets []- mostly used for accessing the properties of an object, or the elements of an array myArray[2] will get the third element in the array.

Braces {}- used during declaration of object literals, or to enclose blocks of code, including function definitions, conditionals, and loops for example.

Double quotes "" - Quotation marks mostly contain strings, a string being the text displayed on a web page as pure text "I am a string". The string must be in quotation marks so it is not read as code.

Single quotes '' - Double and single quotes are interchangeable! As long as they are used consistently, either will work as above. However, with the use of single quotes for a string, the need for an apostrophe makes it more complex. 'This isn't a good string' The second apostrophe here would be read as the end of string and the rest of the text would be lost. This is where the need for backslashes to cancel out would come in to play, for all to be read as one string. 'This isn\'t a bad string'Backslashes are unavoidable if the string needs to incorporate both quotation types, however, if both aren't needed, double quotations eliminate the need to escape.

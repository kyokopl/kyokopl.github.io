Javascript Basics

25 June 2018

How does JavaScript compare to HTML and CSS?

HTML is the language used to provide the basic structure of a website, for example, the text, headings, and images. CSS is the language used to provide the website with style, for example, the colour and styles of fonts and layouts. JavaScript is the programming language that provide the website with a sense of motion, for example, a drop down navigation bar.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the order are task runs. Real life example: Going to the gym - If you are sore, you will stretch. If you are not sore, you will workout. When it comes to loops, they iterate over the same code until the condition turns false. Real life example: Running on the treadmill - Have you hit 500 calories? If no, the action will be to run faster. If yes, the action will be to stop running.

Explain the difference between accessing data from arrays and objects.

Both arrays and objects contain items. The difference is how these are stored. Arrays - items are stored as a list, for example (Kyoko, 27, Wellington) Objects - each item is assigned an ID, for example (Name: Kyoko, Age: 27, Lives in: Wellington) The way in which we access these items is also different. To access object values, we use dot notation. To access array values, we use the numerical index.

Explain what functions are and why they are useful.

A function is used in Javascript to perform a task. We can execute these when a function is called. This can be useful, for example if wanted to know the name of the second person in our contacts list. We could create a function to return this value.
